# REST API
核心概念
1. 互联网上所有可访问的内容，都是自愿
2. 服务器保存自愿，客户端请求自愿
3. 同一个自愿，多种表现形式
4. 参数不同，请求不同自愿
5. 状态转换通过HTTP动词表示
